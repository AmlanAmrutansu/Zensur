1. User registers
2. AI calculates risk score
3. Premium + coverage shown
4. User subscribes

# ⚙️ CORE DECISION ENGINE (CRITICAL)

DO NOT rely only on AI.

---

## Step 1: Trigger Detection

Trigger = TRUE if:

* rainfall > 70 OR
* temperature > 42 OR
* AQI > 400

---

## Step 2: AI Expected Income

expected_income = call AI income service

---

## Step 3: Activity-Based Actual Income Estimate

actual_income = function(active_hours, distance)

---

## Step 4: Activity Drop

activity_drop =
(expected_income - actual_income) / expected_income

---

## Step 5: Work Intent Check

intent_score =
(active_hours * 0.5) + (distance * 0.3)

Reject if:
intent_score < threshold

---

## Step 6: Fraud Check

Call fraud API

Reject if:
fraud_score > 0.7

---

## Step 7: Payout Calculation

loss = expected_income - actual_income

IF loss <= 0 → payout = 0

IF loss < deductible → payout = 0

ELSE:
payout = min(loss - deductible, payout_cap)
